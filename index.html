Parliamo dei componenti!

ma cos'è un COMPUTER?!

Un computer, in italiano elaboratore o calcolatore, è una macchina automatizzata programmabile in grado di eseguire sia complessi calcoli matematici (calcolatore) sia altri tipi di elaborazioni dati (elaboratore).

RAM

È la memoria che contiene il programma in esecuzione

CPU

È l'unità centrale di elaborazione

SCHEDA MADRE

Abbreviata mobo, è un tipo di scheda elettronica principale

DISSIPATORE

Consente l'abbassamento della temperatura della CPU

SSD

Dispositivo di memoria di massa allo stato solido

HDD

È un dispositivo di memoria di massa di tipo magnetico

CASE

Contenitore dentro cui ci sono componenti di un computer

GPU

Componente hardware che elabora il segnale video

ALIMENTATORE

È l'elemento che fornisce energia al PC
